# 06_documentation

## Purpose

Project documentation, reports, and hardware output archives.

## Files

- **`project_report.md`** - Detailed project report
- **`user_manual.md`** - User manual for the system
- **`PERFORMANCE_OPTIMIZATION.md`** - Performance optimization guide
- **`presentation_outline.md`** - Presentation outline
- **`results.txt`** - Test results summary
- **`generate_pptx.py`** - Script to generate presentation slides

## Directory Structure

- **`hardware_output/`** - Archived hardware IP core drivers and documentation from previous builds

## Note
